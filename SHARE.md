# 로컬 페이지를 팀원과 공유하는 방법

로컬에서 돌리는 `http://localhost:3000` 페이지를 **같은 네트워크가 아닌 팀원**에게 공유하려면, 로컬 서버를 인터넷에 잠시 노출하는 **터널 URL**을 쓰면 됩니다.

---

## 0. ngrok 가입 및 authtoken 설정 (최초 1회)

ngrok을 **처음 사용할 때만** 아래 순서대로 진행합니다.

1. **https://ngrok.com** 에 접속해 **무료 가입** 후 로그인합니다.
2. 대시보드에서 **Your Authtoken** (또는 **Getting Started → Connect your account**) 메뉴로 이동해 **authtoken**을 복사합니다.
3. 이 프로젝트 폴더에서 **SETUP_NGROK.bat**을 실행합니다.
4. 화면 안내에 따라 복사한 **authtoken**을 붙여넣고 Enter를 누릅니다.
5. “설정이 완료되었습니다” 메시지가 나오면 이후에는 **SHARE.bat**만 실행하면 됩니다.

> authtoken은 이 PC에만 저장되며, 한 번 설정해 두면 다시 입력할 필요 없습니다.

---

## 1. 공유 URL 생성 (ngrok)

### 1단계: 서버 실행

1. **START.bat**을 더블클릭해 실행합니다.
2. 터미널에 `http://localhost:3000` 이 뜨고, 브라우저에서 접속이 되면 준비 완료입니다.
3. 이 창은 **닫지 말고** 그대로 둡니다.

### 2단계: 공유 URL 만들기

1. **(최초 1회)** ngrok authtoken을 아직 설정하지 않았다면 위 **0. ngrok 가입 및 authtoken 설정**을 먼저 진행합니다.
2. **새 명령 프롬프트(또는 터미널)**를 엽니다.
3. 이 프로젝트 폴더로 이동한 뒤 **SHARE.bat**을 실행합니다.
   ```bat
   SHARE.bat
   ```
   또는 직접 실행:
   ```bash
   npx ngrok http 3000
   ```
4. 터미널에 다음처럼 **공개 URL**이 출력됩니다.
   ```
   Forwarding   https://abc123def.ngrok-free.app -> http://localhost:3000
   ```
5. 이 **https://...ngrok-free.app** 주소를 팀원에게 전달하면, 팀원은 그 주소로 접속해 현재 로컬 페이지를 볼 수 있습니다.

### 3단계: 팀원 접속

- 팀원은 브라우저에서 위 **https** 주소만 입력하면 됩니다.
- **주의**: START.bat(서버)을 끄면 해당 URL로는 접속할 수 없습니다. 공유하는 동안에는 서버를 켜 두어야 합니다.

---

## 2. 참고 사항

| 항목 | 설명 |
|------|------|
| **URL 유효 시간** | ngrok 무료 버전은 **SHARE.bat(ngrok)을 실행할 때마다** URL이 바뀝니다. 터널을 끄지 않으면 그동안은 같은 URL로 접속 가능합니다. |
| **ngrok 가입** | [ngrok 가입](https://ngrok.com) 후 **SETUP_NGROK.bat**으로 authtoken을 설정하면 무료 한도 내에서 더 안정적으로 사용할 수 있습니다. (최초 1회) |
| **보안** | 공유 URL은 누구나 알면 접속할 수 있으므로, 테스트/검토용으로만 사용하고 비공개 정보는 넣지 않는 것이 좋습니다. |
| **대안** | 회사 VPN이나 내부 도메인이 있으면, `내 PC IP:3000` 주소를 공유하는 방식으로도 팀원 접속이 가능합니다. (같은 네트워크 또는 VPN 필요) |

---

## 3. 한 줄 요약

1. **(최초 1회)** **SETUP_NGROK.bat** 실행 → ngrok.com 가입 후 authtoken 붙여넣기  
2. **START.bat** 실행 → 서버 유지  
3. **SHARE.bat** 실행 → 나오는 **https://...ngrok...** 주소를 팀원에게 전달  
4. 팀원은 그 주소로 접속
